# Mini-Build-Web Tutorial Generator

## Project Overview
**Project Name:** Mini-Build-Web  
**Type:** AI-Powered Web Development Tutorial Generator  
**Core Functionality:** Generates step-by-step tutorials for building web technologies from scratch (React, Vue, Express, etc.)  
**Target Users:** Web developers learning frontend/backend technologies

---

## Features
- CLI interface with colored output
- Generates HTML/CSS/JS tutorials
- Framework-specific tutorials (React, Vue, Angular)
- Backend integration tutorials (Node.js, Express)
- Progress tracking
- Markdown output

---

## Tech Stack
- Python 3.x
- LLM API (DeepSeek/OpenAI/Anthropic)
- Markdown generation
